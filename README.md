Vehicle Rental Management System

Overview
This project is a terminal-based vehicle rental management system developed in Python.
It allows a rental agency to manage vehicles, customers, reservations, billing, and reports
using local JSON files.



Features
- Vehicle inventory management
- Customer registration and login (PIN-based)
- Reservation creation with availability checks
- Rental check-in / check-out and invoice calculation
- Data persistence with timestamped backups
- Fleet utilization and revenue reports
